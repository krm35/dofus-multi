isCellUsed(args) → {boolean}

Check if any character is present on the specified cell
Parameters:
Name Type Description
args Object The function arguments
Properties
Name Type Description
cell number The cell number to check for character presence
Returns:
True if someone is on the cell, false if the cell is empty
Type
boolean
Examples
// Check if a cell is occupied before moving there
if (!isCellUsed({cell: 123})) {
    await moveToCell({cell: 123});
} else {
    console.log("Cell 123 is occupied");
}
// Find an empty cell around your position
const adjacentCells = [120, 121, 122, 123, 124];
const emptyCells = adjacentCells.filter(cell => !isCellUsed({cell}));
if (emptyCells.length > 0) {
    await moveToCell({cell: emptyCells[0]});
}
// Check before casting area spells to avoid friendly fire
const targetCell = 456;
if (isCellUsed({cell: targetCell})) {
    console.log("Someone is on target cell, choosing different spell");
    await spell({id: singleTargetSpellId, cell: targetCell});
} else {
    await spell({id: aoeSpellId, cell: targetCell});
}
// Validate positioning before group actions
const gatherPoint = 789;
if (isCellUsed({cell: gatherPoint})) {
    console.log("Gather point is occupied, waiting...");
    await new Promise(resolve => setTimeout(resolve, 1000));
}

(async) moveToCell(args) → {Promise.<boolean>}

Move to a specific cell on the map
Parameters:
Name Type Description
args Object The function arguments
Properties
Name Type Attributes Default Description
cell number <optional>
 Cell number to move to. If not provided, a random cell is chosen
escapeMobs boolean <optional>
 false If true, avoids cells with mob aggression zones (optional)
Throws:
Throws error if cell is undefined/null while in fight
Type
Error
Returns:
Promise that resolves to true if movement succeeds, false otherwise
Type
Promise.<boolean>
Examples
// Move to a specific cell
const success = await moveToCell({cell: 123});
if (success) console.log("Movement successful");
// Move while avoiding mobs
await moveToCell({cell: 456, escapeMobs: true});
// Move to random cell (when not in fight)
await moveToCell({}); // cell will be chosen randomly
// Safe movement with error handling
try {
    await moveToCell({cell: 789, escapeMobs: true});
} catch (error) {
    console.error("Movement failed:", error.message);
}

(async) spell(args) → {Promise.<void>}

Cast a spell
Parameters:
Name Type Description
args Object The function arguments
Properties
Name Type Attributes Default Description
id number The spell ID to cast - See http://localhost:8081/spellsRetro
cell number <optional>
 Target cell to cast the spell on. If not provided, casts on your own cell
t number <optional>
 Time to wait before casting the spell in milliseconds (optional)
retry boolean <optional>
 false Whether to retry the spell if it fails critically (optional)
Returns:
Promise that resolves when the spell is cast
Type
Promise.<void>
Examples
// Cast a spell on your own position
await spell({id: 123});
// Cast a spell on a specific target cell
await spell({id: 456, cell: 234});
// Cast a spell with a delay (useful for timing in combat)
await spell({id: 789, cell: 345, t: 1000}); // Wait 1 second before casting
// Cast a spell with retry on critical failure
await spell({id: 111, cell: 222, retry: true});
// Complete example with all parameters
await spell({
    id: 999,        // Spell ID
    cell: 567,      // Target cell
    t: 500,         // 500ms delay
    retry: true     // Retry on critical failure
});
// Common combat sequence with timing
await spell({id: 123, cell: enemyCell, t: 200});
await spell({id: 456, cell: allyCell, t: 300, retry: true});
